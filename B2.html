Reaktionszeit-Experiment

Sie werden drei unterschiedliche Tests durchführen. Jeder Test wird 30x durchgeführt.
2. Test:
Anleitung: Am Bildschirm werden geometrische Formen angezeigt.
  • Wenn ein Kreis angezeigt wird: Drücken Sie Nichts. Der Kreis wird nach 3 Sekunden verschwinden.
    • Wenn Sie nichts drücken bis der Kreis verschwindet, zählt diese Zeit nicht in ihre Schnitte rein. Es zählt jedoch als Wiederholung.
    • Wenn Sie trotzdem eine Taste drücken, wird diese Zeit registiert und wird als Fehler gezählz
  • Wenn ein Dreieck angezeigt wird:
    • Drücken Sie so schnell wie möglich die LEERTASTE
    • Sollten Sie nicht die LEERTASTE innerhalb von 3 Sekunden drücken, zählt diese Zeit als Reaktionszeit.
  • Wenn Nichts angezeigt wird und sie trotzdem die LEERTASTE drücken, zählt dies als Fehler, jedoch nicht als Wiederholung.

Drücken Sie die LEERTASTE um das Experiment zu starten!
